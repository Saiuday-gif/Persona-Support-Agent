# API Troubleshooting Guide

If you encounter errors while using our API, please follow these steps:

- **401 Unauthorized:** Ensure your API key is valid and has not expired.
- **403 Forbidden:** You do not have permission to access this resource. Check your scope settings.
- **404 Not Found:** Verify the API endpoint URL.
- **429 Too Many Requests:** You have hit the rate limit. Wait a few seconds before retrying.
- **500 Internal Server Error:** This is an issue on our side. Please contact support@company.com with the request ID.
